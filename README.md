# 🧠 Portafolio Académico: Inteligencia Artificial y Ciencia de Datos

Bienvenido a mi repositorio personal. Aquí almaceno, sincronizo y gestiono el progreso de mis proyectos universitarios, prácticas de laboratorio e investigaciones en el campo de la Inteligencia Artificial y Data Science.

Este espacio sirve como mi entorno de trabajo distribuido entre mis diferentes estaciones de trabajo (Laptop/Desktop) y como historial de mi evolución como desarrollador.

## 🛠 Tecnologías y Herramientas
Principales tecnologías exploradas en este repositorio:
* **Lenguajes:** Python, R, SQL.
* **Librerías:** Pandas, NumPy, Scikit-learn, TensorFlow/PyTorch, Matplotlib.
* **Entornos:** Jupyter Notebooks, VS Code.
